# Aryan's Portfolio

A minimalist, high-performance portfolio built with Next.js 15, TypeScript, and Vanilla CSS. Designed to be fast, responsive, and visually striking.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Vanilla CSS (Custom Glassmorphism)
- **Language**: TypeScript
- **Icons/Fonts**: Geist (via `next/font`)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the result.

## Design Philosophy

- **Minimalist**: Focus on typography and hierarchy.
- **Premium**: Modern color palettes and subtle micro-interactions.
- **Performance**: Zero-dependency styling for lightning-fast loads.
